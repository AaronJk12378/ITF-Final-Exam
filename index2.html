FINAL EXAM QUESTIONS

1. Which of the following best describes a "web browser"?

B. A software application for retrieving and displaying web content

2. In the context of networking, what is the primary purpose of a router?

B. To connect multiple networks and direct data traffic between them

3. What describes the evolution from Web 1.0 to Web 2.0?

A. From static pages to user-generated content and social media

4. Which device forwards data packets between computer networks using routing protocols?

C. Router

5. What is a web browser used for?

C. It retrieves and displays internet content such as pages, images, and videos
